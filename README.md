# GPU Proot Acceleration

This repository contains a script to enable GPU acceleration (OpenGL over Vulkan via Zink) in Proot-Ubuntu / udroid environments on Android.

## Installation
```bash
curl -Ls https://raw.githubusercontent.com/jaydenmayers2436/gpu-proot-accel/main/zink-accel.sh | bash
```

## Requirements
- Android 13+
- Termux installed
- udroid or any Proot-Ubuntu distribution
- Vulkan drivers installed on the host (Termux)
